fetcher (데이터 분석) 네이버증권+yfinance사용했음

가상환경 키고
pip install -r requirements.txt
pip install yfinance beautifulsoup4 lxml requests
python manage.py migrate
npm install (frontend설치)

팀원 대시보드  → http://localhost:5173/
내 스크리너  → http://localhost:5173/screener


pip 패키지
─────────────────────────────
requirements.txt 안에 있는 것들
yfinance
beautifulsoup4
lxml
requests

─────────────────────────────
